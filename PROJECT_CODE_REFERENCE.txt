ROLE: Main Flask application — routes, auth, admin/student flows, QR API, CLI.

--- Standard library imports ---

  flask.render_template      Render Jinja2 HTML templates.
  flask.request              Form data, JSON body, query args, headers.
  flask.send_file            Stream PNG and CSV downloads.
  flask.session              Signed cookie session (user_id, csrf_token).
  flask.url_for              Reverse URL generation for redirects and templates.
  werkzeug.security.check_password_hash   Verify login password.

--- Local project imports ---

--- Global / module-level objects ---
  app                        Flask(__name__); main WSGI application (gunicorn: app:app).
  _instance                  Path string: Flask app.instance_path.
  (side effect) os.makedirs(_instance, exist_ok=True)  Ensures instance folder exists.

--- Functions & decorators (definitions) ---

  @app.context_processor
  inject_current_user()
      Returns dict {"user": current_user()} for all Jinja templates.

  _refresh_csrf() -> None
      Sets session["csrf_token"] to secrets.token_hex(16).

  _validate_csrf() -> bool
      Compares form field csrf_token or header X-CSRF-Token to session value.

  login_required(fn)
      Decorator: if no session["user_id"], flash + redirect to login with ?next=.

  admin_required(fn)
      Decorator: requires login + User.role == "admin"; else redirect student_dashboard.

  current_user() -> User | None
      Loads User by session["user_id"] or returns None.

  build_qr_payload(session_id: str) -> str
      Returns "session_id|unix_timestamp_utc" for embedding in QR.

  parse_qr_payload(raw: str) -> tuple[str, int] | None
      Parses QR string into (session_id, ts); None if invalid.

  validate_qr_scan(session_id: str, embedded_ts: int) -> tuple[bool, str]
      Checks LectureSession exists and |now_ts - embedded_ts| <= QR_VALIDITY_SECONDS.

  @app.cli.command("init-db")
  init_db_command()
      Flask CLI: db.create_all() inside app context; prints confirmation.

  @app.cli.command("create-admin") + @click.option(...) x3
  create_admin_command(email, password, name)
      Flask CLI: inserts User with role admin if email not taken.

  --- Route view functions (Flask @app.route) ---

  index()                         GET  /          Redirect by role or to login.
  login()                         GET/POST /login
  register()                      GET/POST /register
  logout()                        GET  /logout
  admin_dashboard()               GET  /admin
  admin_create_session()          POST /admin/session/create
  admin_session_qr(session_id)    GET  /admin/session/<session_id>
  admin_session_qr_image(session_id) GET /admin/session/<session_id>/qr.png
  admin_attendance()              GET  /admin/attendance
  admin_export_csv()              GET  /admin/export.csv
  student_dashboard()             GET  /student
  student_mark()                  POST /student/mark   (JSON API for scanner)

  --- Error handlers ---

  not_found(e)                    @app.errorhandler(404) → errors/404.html
  server_error(e)                 @app.errorhandler(500) → errors/500.html

--- Startup block ---
  with app.app_context(): db.create_all()
      Ensures tables exist when the app module loads (SQLite or PostgreSQL).

--- Inner functions (closures) ---
  login_required.wrapper        Checks session user_id before calling route.

--- Variables of note inside routes ---

2. FILE: config.py
================================================================================
ROLE: Central configuration from environment variables; SQLite path helper.

--- Standard library ---
  os                         getenv, path join, makedirs, dirname, abspath.
  datetime.timedelta         PERMANENT_SESSION_LIFETIME for Flask sessions.

--- Module-level ---
  BASE_DIR                   Absolute path to directory containing config.py.

--- Functions ---
  _normalize_database_url(url: str) -> str
      Rewrites postgres:// → postgresql:// for SQLAlchemy 2.x.

--- Class ---
  class Config
      Class body assigns (at import time):
        SECRET_KEY                    From env or dev default string.
        SQLALCHEMY_DATABASE_URI       DATABASE_URL normalized, OR SQLite file URI.
        SQLALCHEMY_TRACK_MODIFICATIONS False (Flask-SQLAlchemy recommendation).
        PERMANENT_SESSION_LIFETIME    timedelta from SESSION_TIMEOUT_MINUTES (default 60).
        QR_VALIDITY_SECONDS           int from env (default 300).
        ADMIN_REGISTRATION_SECRET     str from env or None if empty.

      Side effect: if no DATABASE_URL, creates BASE_DIR/instance/ and points SQLite there.

3. FILE: models.py
================================================================================
ROLE: Database schema as SQLAlchemy ORM models.

--- Standard library ---
  uuid                       uuid.uuid4() for string session primary keys.
  datetime.datetime          Default factory for LectureSession.created_at.

--- Third-party ---
  flask_sqlalchemy.SQLAlchemy  db object (bound to app in app.py).

--- Module-level object ---
  db                         SQLAlchemy() extension instance.

--- Classes (ORM models) ---

  class User(db.Model)
      __tablename__ = "users"
      Columns: id, name, email, password_hash, role.
      Relationships:
        sessions_created   → LectureSession (via admin_id / creator backref).
        attendance_records → Attendance (user backref).

  class LectureSession(db.Model)
      __tablename__ = "lecture_sessions"
      Columns: id (str UUID PK, default on insert), title, admin_id (FK users.id),
               created_at.
      Relationship: attendance_rows → Attendance (lecture_session backref).

  class Attendance(db.Model)
      __tablename__ = "attendance"
      Columns: id, user_id (FK), session_id (FK lecture_sessions.id),
               date, time.
      __table_args__: UniqueConstraint(user_id, session_id) name uq_attendance_user_session.

4. FILE: static/js/scanner.js
================================================================================
ROLE: Client-side QR scanning and POST to Flask; IIFE (no global exports).

--- External dependency (not in this file; loaded by template) ---
  Html5Qrcode                 Constructor from html5-qrcode.min.js (unpkg CDN).
  Html5Qrcode.getCameras()    Enumerate video input devices.

--- Browser APIs used ---
  document.getElementById     Get #reader, #btnStart, #btnStop, #scanMessage.
  document.querySelector      meta[name="csrf-token"] for CSRF value.
  meta.getAttribute("content")
  fetch(url, options)         POST /student/mark with JSON + headers.
  JSON.stringify              Body { qr_text: "..." }.
  Response.json / Promise chains  .then / .catch / .finally.
  addEventListener("click")   Start / stop camera buttons.

--- IIFE-scoped variables ---
  readerEl, btnStart, btnStop, msgEl, meta, csrf
  html5QrCode                 new Html5Qrcode("reader")
  running, busy               Prevent double-submit / double-start.

--- Inner functions ---
  setMessage(text, kind)      Updates #scanMessage text and ok/err CSS class.
  submitQr(qrText)            fetch POST to /student/mark with X-CSRF-Token.

--- Event handlers (anonymous functions) ---
  btnStart click: Html5Qrcode.getCameras → html5QrCode.start(cameraId, config, onSuccess, onError)
      onSuccess decodes QR → submitQr(decodedText).
  btnStop click: html5QrCode.stop() → reset buttons and message.
